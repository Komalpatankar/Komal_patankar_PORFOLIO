Quizzical: AI Quiz Generator

Quizzical is an AI-driven adaptive quiz generator built with Figma, React.js, Tailwind CSS, Gemini API, and MongoDB. It generates quizzes from documents or prompts using AI and vector database integration, with adaptive difficulty based on user performance. Key features include real-time keyword-based quiz generation, quiz editing, multi-user collaboration, and Google Forms integration for sharing and email notifications. I led the UI/UX design, focusing on responsive, intuitive user experiences that support personalized learning.

INSUREIQ: LIC Chatbot

INSUREIQ, a multilingual chatbot for LIC policy support. I built the full system using a RAG pipeline with LangChain, FAISS, and Google Gemini LLMs. The chatbot retrieves relevant LIC document chunks and generates accurate, cited responses in English and Hindi. I also developed a user-friendly interface using Streamlit. This project helped me apply NLP, vector search, and LLMs to solve a real-world problem with a focus on accessibility and factual accuracy.
